How does this page work?

This page is built entirely from the Markdown files in this repository.

  • Jekyll scans the repo for every .md file (except README.md) and injects them into the page at build time.
  • The search box filters that list instantly by title or filename—no external data sources or wiki pages.
  • Click any card to open the Markdown directly from this repo in the built-in reader.

If you see an empty list locally, run Jekyll (bundle exec jekyll serve) or publish to GitHub Pages so the page can index the files.

GitHub Pages builds the article list for you so every Markdown file in this repository is searchable and readable here.
